# Australian Cuisine Restaurant Website

A modern, performance-focused restaurant website built to establish a premium brand presence, drive dine-in reservations, and promote upcoming events such as seasonal and special-day offers.

This project is developed for a Sri Lankan restaurant introducing Australian cuisine to the local market.

---

## Tech Stack

- **Framework:** Next.js
- **Styling:** CSS / Tailwind / Styled Components
- **Animations:** Framer Motion
- **Forms:** Custom reservation request flow

---

## Key Features

- Fully responsive design (mobile-first)
- Custom reservation system with availability view
- Event promotion section on the homepage
- Smooth page transitions and subtle UI animations
- SEO-friendly structure and fast load times
- External ordering links (Uber Eats & PickMe)
